Goal

Further explore the class project data with an eye towards making the data better.

Objectives

Write well-formatted queries to answer the questions found in DETAILS below
Explain what is wrong with the two queries found in Section #2 of the DETAILS
What data would you need to populate a chart? (See Section #3 of the DETAILS)

Details

1. Answer these questions with queries

  • List the assigned school and all school choices for each person, ordered by the assigned school.
  • How many times was each school chosen as a first choice? Order the output by the number of times chosen
  • Translate the school names from their two-character abbreviation to the full name. Use ch1schna, and only list each school once. HINT 1; HINT 2: Booker T Washington = BTW = WA
  • Advanced: If you were one of the parents that chose Bottenfield as your second choice, what was the chance that you were assigned to Bottenfield?
    HINT 1: For percentage, use the formula:
    100*(times assigned)/(total time chosen)
  • Advanced (optional): If instead you were one of the parents that chose Doctor Howard as your second choice, what was the chance that you were assigned to Howard? Second question/query: Why? (HINT: ch1stat)

2. Explain why these queries do not correctly answer the Advanced question about Bottenfield:
Query 1 

 SELECT 100 * count(asgsch) / count(*) from lottery where ch2schna = 'BO';

Query 2 

 SELECT 100 * count(case when asgsch = 10 then 1 else 0 end) / count(*) from lottery where ch2schna = 'BO';

You are welcome to come up with queries if you like, but that is not required for this stage. Tell me which columns, which rows, and any sort of aggregation (ie, sum? max?) you would need.
