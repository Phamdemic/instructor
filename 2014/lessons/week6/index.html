Week 6

Goal

As an introduction to what you will be doing for the Midterm next week, you will tackle tackle one question; your task is to answer this question with a single query, outputing the answer in JSON format including the full name of each school. The question is:


What are my chances of getting into any given school (based on first choices only)? Sort the output from highest chance to lowest chance.

Here is an example one your fellow students suggested:
So 85 people were assigned BO out of 89 who had them as a first choice. So is the probability for BO is 85/89?
Answer: Yes

Objectives

Use the "think a query problem" method from Chapter 6; document each step.
Step 1: The Data You Are Working With
Identify all the data we need to address this question
Step 2: Consider Aggregates as Separate Queries
Will you need an aggregate function (ie, sum)? If so, what data needs to be grouped? What keywords from the question give you a hint?
Step 3: The Main Filter
How can we reduce the number of rows?
Step 4: Core Joins
What columns will join you join multiple tables on? Consider also subqueries.
Step 5: Apply Polish
This is where you format the JSON, just like in #6 of Chapter 5. For example:
['School 1', 23.7],
['School 2', 16.1],
['School 3', 14.2],
['School 4', 14.0],
['School 5', 12.5],
['School 6', 12.1],
['School 7', 11.8],
['School 8', 11.7],
['School 9', 11.1],
['School 10', 8.9],
['School 11', 8.9],
